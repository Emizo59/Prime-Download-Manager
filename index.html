تنبيه هام بخصوص الحماية في ويندوز / Windows Smart App Control & Defender Notice
نظراً لعدم توفر شهادة رقمية تجارية للبرنامج حالياً، قد يظهر تنبيه من ميزة Smart App Control أو Windows Defender. إذا واجهت مشكلة في التثبيت، يرجى إيقاف Smart App Control مؤقتاً أو اختيار "More Info" ثم "Run Anyway" لإتمام التثبيت بأمان.
Since the application currently does not have a commercial code-signing certificate, Windows 11 Smart App Control (SAC) or Windows Defender might flag the installer as untrusted. If you get a block message, please temporarily disable Smart App Control or click "More Info" -> "Run Anyway" in Defender to complete the installation.

طريقة فتح البرنامج على ماك (أول مرة) / How to open on macOS (first time)
البرنامج غير موقّع بشهادة Apple، فأول مرة ماك ممكن يقول "cannot be opened". لتشغيله: افتح الـ DMG واسحب البرنامج إلى Applications، بعدين كليك يمين على البرنامج → Open → Open. أو من System Settings ← Privacy & Security اضغط Open Anyway. بتعملها مرة واحدة بس، وبعدها بيفتح عادي بالدبل-كليك. (فيه ملف "READ ME FIRST" جوّه الـ DMG بيشرح ده بالعربي والإنجليزي.)
The app isn't notarized by Apple, so macOS may say it "cannot be opened" the first time. To run it: open the DMG and drag the app into Applications, then right-click the app → Open → Open. Or go to System Settings → Privacy & Security and click Open Anyway. You only do this once — after that it opens normally with a double-click. (A "READ ME FIRST" file inside the DMG explains this in Arabic & English.)
